LdsIo.cache.destroy().then(function () {
  location.reload();
});

You are in live mode. Set your API source to the beta site and refresh to enter developer mode. 

LdsIo.cache.destroy().then(function () {
  return LdsIo.storage.set('providerUri', 'https://beta.lds.io');
}).then(function () {
  location.reload();
});
